# HomeEase – Home Services Booking System

## 📌 Project Description

HomeEase is a full-stack web application that connects customers with service providers for booking home services like plumbing, electrical work, cleaning, painting, and repairs. It provides a smooth and user-friendly experience for managing bookings and services online.

## 🚀 Features

* User Registration & Login (JWT Authentication)
* Role-based access (Admin, Provider, Customer)
* Browse and book home services
* Provider dashboard to manage services & bookings
* Admin panel for managing users and services
* Booking status tracking
* Responsive UI

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Vite
* Tailwind CSS

**Backend:**

* Java
* Spring Boot
* Spring Security
* REST APIs

**Database:**

* MySQL

## 📂 Project Structure

* frontend/ → React frontend
* backend/ → Spring Boot backend

## ⚙️ Setup Instructions

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
mvn clean install
mvn spring-boot:run
```

## 🔐 Authentication

Uses JWT-based authentication for secure login and role management.
